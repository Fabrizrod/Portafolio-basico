Mini Proyectos

Numero Secreto - aquí

    Un juego interactivo donde el usuario debe adivinar un número aleatorio generado por el sistema. Cada intento recibe pistas para acercarse a la respuesta correcta. Perfecto para practicar lógica y manipulación del DOM con JavaScript.

Amigo Secreto - aquí

    Una aplicación web que facilita la organización de un sorteo de Amigo Secreto. Los participantes ingresan sus nombres, y el sistema se encarga de asignar aleatoriamente a cada persona su Amigo Secreto sin repeticiones, garantizando que nadie se asigne a sí mismo.
